# Inventory Management System

A web-based Inventory Management System developed using PHP and MySQL to help businesses manage products, suppliers, stock levels, and inventory records efficiently.

## Features

### Product Management

* Add new products
* Edit product information
* Delete products
* Generate and manage unique SKUs

### Inventory Tracking

* Monitor stock levels
* Update inventory records
* View product availability

### Category Management

* Organize products into categories
* Filter products by category

### Supplier Management

* Manage supplier information
* Associate products with suppliers

### Search & Filter

* Search products by name or SKU
* Filter products by category and supplier

### Dashboard

* View inventory records in a structured table
* Quick access to product operations

## Installation

### Prerequisites

* PHP 8.x
* MySQL
* XAMPP

### Setup

1. Clone the repository

git clone https://github.com/Dj7517/inventory-management-system.git

2. Move the project folder to:

C:\xampp\htdocs\

3. Start Apache and MySQL from XAMPP.

4. Create a database named:

inventory_db

5. Import the SQL file into MySQL.

6. Configure database credentials in:

config/Database.php

7. Open your browser and visit:

http://localhost/inventory-system
